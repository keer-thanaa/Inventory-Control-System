#  Inventory Control and Stock Management System

A simple Java-based Inventory Control and Stock Management System developed using Object-Oriented Programming (OOP) concepts.

##  Features
- Add, update, delete, and view products.
- Displays complete stock information.
- Prevents duplicate entries.
- Designed using clean and modular OOP principles.

##  Technologies Used
- **Language:** Java  
- **IDE:** Eclipse / IntelliJ IDEA / Command Line  
- **Concepts:** Classes, Objects, Encapsulation, File Handling
